# KeenKeeper

KeenKeeper is a friendship tracker app that helps users remember when to reconnect with important people. It provides friend health status, quick check-ins, a timeline, and interaction analytics.

## Technologies Used

- React.js
- Vite
- React Router DOM
- Recharts
- React Toastify
- React Icons
- CSS (responsive layout)

## Key Features

- Home dashboard with banner, summary cards, and responsive friend grid
- Friend details page with quick check-in buttons (Call, Text, Video) that log timeline entries and show toast notifications
- Timeline page with interaction history and filter options by interaction type
- Friendship analytics page with a Recharts pie chart for Call/Text/Video counts
- Custom 404 page and SPA deployment rewrite support

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
